Scandic API Days – Call for Papers

About

API Days Scandinavia is part of global series of API Days events organized around the world. For the first time in history Scandinavian API community is gathered in one event to share knowledge, good practices, success stories and insights of emergent API economy.

Event is organized by API:Suomi (Finnish API community) with support of API Days global network, COSS (the Finnish Centre for Open Systems and Solutions), API Duo OÜ (API strategy consulting) and Tredea (Tampere Region Economic Development Agency). API Days Scandinavia is 2-day event and will be organized in Tampere, Finland.

Tampere is the second-largest urban area and third most-populous (~255 000 habitants) individual municipality in the country, after Helsinki and Espoo municipalities (2014 data). It's also the most populous Finnish city outside the Greater Helsinki area and inner Finland's major urban, economic and cultural hub.

The City of Tampere is running for the title of Europe’s innovation capital in the contest ”The European Capital of Innovation Award – iCapital”, organized by the European Commission. Tampere is growing city with a lot of companies focusing in ICT-development. Tampere is also home for multiple heavy industry giants like KoneCranes and Kalmar. These companies are pioneers in Internet of BIG things. In addition Tampere based ICT- companies have strong foothold in developing and maintaining national APIs driven data exchange layer (kansallinen palveluväylä) in Finland.

For these reasons, event focus will be in heavy industry and goverment maintained information exchange layers such as X-Road which is in production both in Finland and Estonia.

Our themes for ‘16

API Days Scandinavia is part of global series of API Days events organized around the world. For the first time in history Scandinavian API community is gathered in one event to share knowledge, good practices, success stories and insights of emergent API economy.

  • Strategy Stream: business imperatives and opportunities afforded by APIs
  • Execution Stream: design, architecture, security, quality and new technologies
  • Community Stream: community building, meetups, show cases and workshops

Session length is 45 minutes including discussions.

Important dates

  • Selected speakers are notified 1.4.2016 by email.

How to submit

Submit your suggestion with Google form, which can be found from here. All information given is publicly used unless otherwise stated. Required information is:

  • CFP title
  • Cfp description (max 150 chars)
  • Full name / name of organization
  • Short bio / introduction about speaker (max 150 characters)
  • Link to presenter photo
  • Contact email (not used publicly)
  • Presentation language: English or Finnish (English is preferred)
  • Cfp categories (choose max 3)
