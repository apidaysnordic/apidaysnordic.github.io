10:00 - 11:00 Onboarding to X-Road with Joint X-Road community –  Karri Niemelä

Finnish and Estonian X-Road communities joined forces last year and decided to establish joint global X-Road community. Karri Niemelä is current Community Manager. Karri will demonstrate how community works and how it has helped businesses to onboard X-Road with tools such as integration patterns with code examples.

14:00 - 15:00 APIOps: APIs and DevOps workshop –  APIOps Finland

DevOps and APIs related workshop to discuss and identify solutions, best practices and possibilities.

15:00 - 16:00 AWS Serverless Workshop –  Mikael Puittinen

The workshop introduces the audience to AWS services such as API Gateway, Lambda functions that enable developers to develop services to the cloud without deploying any servers. the serverless framework that streamlines the process of developing and deploying API Gateway / Lambda based solutions The workshop consists of a short theory introduction and of a step-by-step tutorial how to build a simple blog backend using API Gateway, Lambda and serverless.

Prerequisites for the workshop (for the hands-on excercise):
  • A laptop with Node 4 or 5 installed An AWS account
  • An AWS IAM User with Administrator privileges and keys set up on the laptop AWS CLI recommended but not required
  • Instructions for preparing the environment will be given here later.
